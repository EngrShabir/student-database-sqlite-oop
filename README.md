# student-database-sqlite-oop
Student database system built with Python, SQLite and Object-Oriented Programming (OOP). Full CRUD with interactive menu.
# Student Database — SQLite with OOP

## 📌 Description
A fully object-oriented student database system using Python and SQLite.

## ⚙️ Features
- Add student (name, age, grade)
- View all students
- Update student by ID
- Delete student by ID
- Built using class, __init__, self, and methods

## 🛠️ Tech Used
- Python
- SQLite3
- Object-Oriented Programming (OOP)

## ▶️ Run
```
python main.py
```
